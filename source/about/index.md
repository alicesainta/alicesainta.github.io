## 关于我

你好，这里是 AliceSainta。  
我是一个前端开发者，长期关注「可维护的工程结构」和「有质感的交互体验」。

相比“炫技式页面”，我更在意两件事：

- 做有意思的设计
- 交互是否自然到用户几乎感觉不到学习成本

## 我在做什么

目前主要在做三类事情：

- 用 TypeScript + Vue/React 打磨中小型项目的工程质量
- 复盘前端面试与实战问题，沉淀成可复用的知识卡片
- 结合 AI CLI 工具优化个人开发流（方案、改码、验证一体化）

## 技术栈地图

### 核心栈

- TypeScript
- Vue 3
- React / Next.js
- Node.js（基础服务与脚本能力）

### 工程与协作

- pnpm

- ESLint / Prettier（规范化协作）

### 持续探索

- 前端性能优化与可观测性
- 组件抽象与设计系统落地
- AI 辅助开发工作流

## 这个博客会写什么

这个站点主要记录三类内容：

- 前端工程实践：真实项目里的取舍与坑点
- 面试复盘：高频题的理解框架与表达方式
- 工具工作流：提升效率但不过度依赖“黑箱”

我会尽量写“可落地、可复现”的内容，而不是只给结论。

## 兴趣与审美

技术之外，我也喜欢东方Project，尤其红魔馆相关内容。  
对我来说，好的作品和好的代码有一个共同点：结构稳定，但能让人反复回味。
